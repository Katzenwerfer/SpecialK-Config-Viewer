# SpecialK Config Viewer

This is an extremely WIP project to fetch and visualize all of Special K's ini parameters

Currently I've only developed a PowerShell script to:

1. fetch the [`config.cpp`](https://github.com/SpecialKO/SpecialK/blob/main/src/config.cpp) file
2. extract data from it
3. save it into a CSV file

I plan on eventually doing a web app for this, though no idea when I'll get to it.
